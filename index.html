Wo finde ich diese Werte bei Schwab?

Account Value / NLV: Accounts → Summary → Total Value oben links
Margin Equity: Accounts → Balances → Balance Details → rechte Spalte → Margin Details & Buying Power → Margin Equity
Equity Percent: Direkt darunter → Equity Percent
Maintenance Requirement: thinkorswim → Monitor → Account Statement → Margin Requirements, oder Schwab Web → Balances → Option Details (weiter unten scrollen)
Short Options Buying Power: Accounts → Balances → Balance Details → Link „Margin Buying Power" klicken → Options → Short Options (minimum equity required)
